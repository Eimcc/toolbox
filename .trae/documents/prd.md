## 1. Product Overview
Windows XP 像素风格的工具箱网页应用，提供简洁易用的工具集合。
- 主要用途是提供图片和视频格式转换等实用工具，解决用户日常文件处理需求
- 目标用户为需要快速进行文件格式转换的个人用户，市场价值在于提供免费、易用的在线工具

## 2. Core Features

### 2.1 User Roles (if applicable)
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Normal User | 无需注册 | 使用所有工具功能 |

### 2.2 Feature Module
1. **桌面页面**：Windows XP 风格桌面，包含工具图标，任务栏
2. **图片格式转换工具**：上传图片，选择输出格式，转换并下载
3. **视频格式转换工具**：上传视频，选择输出格式，转换并下载

## 3. Core Process
用户打开应用 → 看到 Windows XP 风格桌面 → 点击工具图标 → 打开对应工具窗口 → 上传文件 → 选择输出格式 → 点击转换 → 下载转换后的文件

```mermaid
graph TD
  A[打开应用] --> B[显示桌面]
  B --> C[点击图片转换图标]
  B --> D[点击视频转换图标]
  C --> E[打开图片转换工具]
  D --> F[打开视频转换工具]
  E --> G[上传图片]
  F --> H[上传视频]
  G --> I[选择输出格式]
  H --> J[选择输出格式]
  I --> K[点击转换]
  J --> L[点击转换]
  K --> M[下载转换后的图片]
  L --> N[下载转换后的视频]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色调：蓝色 (#0054A6)、绿色 (#008000)、灰色 (#C0C0C0)、白色 (#FFFFFF)
- 按钮风格：3D 效果，有凸起和凹陷状态
- 字体：系统默认字体，如 Tahoma 或 Arial
- 布局风格：Windows XP 经典窗口布局，包含标题栏、菜单栏、工作区
- 图标风格：像素风格，模拟 Windows XP 系统图标

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|

### 4.3 Responsiveness
- 设计为桌面优先，适配不同屏幕尺寸
- 在小屏幕设备上，工具窗口会自动调整大小
- 支持触摸操作，确保在平板设备上也能正常使用
